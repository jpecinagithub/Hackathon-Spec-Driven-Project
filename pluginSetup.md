# Setup Cheat Sheet

## Check Git

git --version

## Install Git (Windows only)

https://git-scm.com

## Install Claude Code

Mac:

curl -fsSL https://claude.ai/install.sh | bash

Windows:

irm https://claude.ai/install.ps1 | iex

## Verify Claude Code

claude --version

## Create Project Folder

Mac:

mkdir ~/Desktop/my-hackathon-project
cd ~/Desktop/my-hackathon-project

Windows:

mkdir ~\Desktop\my-hackathon-project
cd ~\Desktop\my-hackathon-project

## Launch Claude

claude

## Add the Marketplace

/plugin marketplace add https://github.com/challengepost/devpost-curriculum

## Install the Plugin

/plugin install hackathon-in-a-plugin@devpost-curriculum

## Reload Plugins

/reload-plugins

## Restart Claude

/exit

claude

## Start the Hackathon

/onboard
